Pacific Coast Trail

The map was Generated using QGIS and the use of Erwin Raisz Landforms of the United States Map. The map was georeferenced to insure that the Erwin Map could access geospacial data from the map referenced on QGIS. Once the map is digitized I can create new points, lines, and polygons. This kind of power allows to use classic maps as base map for our web projects. In the case of the map displayed to left of this paragraph the map has been digitized to display the Pacific Crest Trail from Washington State to California.

Syrian Refugee Distribution by Country

The map was built using CartoDB. It depicts the mass exodus of the Syrians to neighboring countries with registered refugees. The map has color coded the countries to show the percentage of the total refugees (September 2016) within each particular country. The purple circles are scaled proportionally display the population of prominent Syrian Cities. When the user zooms in closer to the map, orange circle will display registered refugee sites. The map does not take into account unregistered refugees within in the region or those that have made their way to Eastern or Western Europe. The map helps to depict the magnitude of the humanitarian crisis and the sheer volume of the Syrian population that has been displaced.

  • See the map: Syrian Refugee Distribution by Country
  • Data Source: The Human Data Exchange

Proportional Symbol Map Displaying Nuclear,
Solar and Hydro Power Plants

This map was built with custom JavaScript code drawing upon the Leaflet JavaScript Library to create a bivariate map. The map displays the location Nuclear(neon-green), Solar(orange) and Hydro(blue) power production plants. The The point data is displayed in proportional symbols based on the power output of each power plant. The viewer can filter the map in several ways. First, the reader can click on the map and it will only display power facilities located within 500 KM of the clicked area. Second, the reader can filter the map by its layers and by unchecking power sources in the layers control box. Additionally, by clicking on individual points a popup window will display the name of the plant, its fuel source and the distance the plant is located in relation to the user's click. This map allows you to see the saturation of the listed fuel sources and their production capacity. bla
